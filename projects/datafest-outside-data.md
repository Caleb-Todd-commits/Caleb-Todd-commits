# DataFest — Best Use of Outside Data

DataFest is a fast-paced data analysis competition where teams work under time pressure to explore a real dataset, find meaningful patterns, and present a clear story.

My team received recognition for **Best Use of Outside Data**.

## What the Award Shows

This award matters because it shows more than basic data cleaning. It demonstrates the ability to improve analysis by adding context from sources beyond the original dataset.

The project showed strengths in:

- identifying what the original dataset was missing
- finding outside data that could add useful context
- combining data sources into a stronger analysis
- turning raw information into a more meaningful explanation
- communicating findings clearly under competition constraints

## Why Outside Data Matters

A dataset rarely tells the whole story by itself. Strong analysis often depends on knowing what additional context can explain patterns, reveal gaps, or challenge assumptions.

Outside data can help answer questions like:

- What external factors might explain the trend?
- Is this pattern local, seasonal, economic, behavioral, or demographic?
- Does the dataset miss a key variable?
- Can a public source strengthen or validate the conclusion?

For cybersecurity and security analyst roles, this matters because analysts often need to combine multiple sources of evidence before making a judgment.

The same thinking applies to:
